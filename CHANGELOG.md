# Changelog

## [1.0.0] — 2026-06-17

### 首个版本 — P0 基础核心功能

- 便签 CRUD（新增/编辑/删除/撤销删除/完成/取消完成）
- 手动置顶 + 标签置顶（最多 3 个置顶标签）
- 手动拖拽排序
- 标签管理（CRUD + 置顶 + 批量删除）
- FTS5 全文搜索 + LIKE 中文兜底
- 搜索筛选（关键词 + 标签 + 时间范围）
- 前端实时校验 + 后端提交校验
- SQLite 数据持久化
- 冒烟测试 + UI 测试通过

> **注意：** 当前为基础版本，UI 较粗略，仍有大量优化和修正空间。详见 spec.md 和后续版本计划。
